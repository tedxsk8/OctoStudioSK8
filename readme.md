# Landing Page 改版需求

請分析目前專案中的 Landing Page 結構與元件，並在保留現有技術架構、樣式系統與響應式設計的前提下，重新調整內容與版面配置。

## 專案目標

此 Landing Page 用於招生推廣：

「OctoStudio 滑板小子程式冒險營」

目標受眾：

* 家長（主要決策者）
* 大班或國小 1~3 年級學生（實際參與者）

主要轉換目標：

* 點擊報名按鈕
* 加入 LINE 官方帳號
* 填寫報名表單
* 預約體驗課

---

## 設計風格

請維持：

* 活潑、科技感、創意感
* 滑板文化元素

# UX 要求

請優先考量：

1. 手機版體驗
2. 滾動式閱讀節奏
3. CTA 重複出現
4. Hero 區域轉換率
5. 家長快速理解課程價值

# 技術要求

請不要大幅破壞既有架構。

優先：

* 重用現有 Components
* 重用既有 CSS / Tailwind
* 保留現有 Router
* 保留既有動畫系統

如果缺少元件再新增。

最後請：

1. 說明修改策略
2. 列出新增元件
3. 列出修改元件
4. 提供完整修改後程式碼
5. 說明檔案異動清單
